# Data Directory

Place your `creditcard.csv` dataset here.

## Download Dataset

1. Visit: https://www.kaggle.com/mlg-ulb/creditcardfraud
2. Download the dataset (requires Kaggle account)
3. Extract `creditcard.csv`
4. Place it in this directory

## Dataset Information

- **Name**: Credit Card Fraud Detection
- **Source**: Kaggle (ULB Machine Learning Group)
- **Size**: ~150 MB
- **Transactions**: 284,807
- **Fraud Cases**: 492 (0.172%)
- **Features**: 30 (Time, Amount, V1-V28, Class)
- **Format**: CSV

## Privacy

The features V1-V28 are already PCA-transformed for privacy protection. No actual card numbers or personal information are present.
